# ❤️ Heart AI Platform

An interactive web-based platform for analyzing heart disease risk factors and visualizing cardiovascular health data.

The platform provides a modern dashboard interface where users can explore medical parameters, analyze heart disease risk factors, and visualize dataset insights.

---

## 🌐 Live Demo

View the project online:

https://jahanvi88.github.io/heart-ai-platform/

---

## 📊 Project Overview

Heart disease is one of the leading causes of death worldwide.  
This platform helps visualize and analyze cardiovascular health indicators using healthcare datasets.

The project demonstrates how web technologies can be used to build a modern healthcare analytics interface.

---

## ✨ Features

- Responsive modern UI
- Heart disease risk analysis interface
- Interactive dataset visualization
- Medical parameter input form
- Image-based health factor explanations
- Clean dashboard layout
- Bootstrap responsive design
- Animated UI components
