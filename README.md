# ❤️ Heart Disease Analysis and Prediction System

This project is a web-based application that helps analyze patient medical data and predict the possibility of heart disease.

The system uses medical attributes such as:

- Age
- Blood Pressure
- Cholesterol
- Chest Pain Type
- Maximum Heart Rate
- Blood Sugar Levels

These parameters are used to analyze the risk of cardiovascular disease.

---

## 🚀 Features

- Heart disease prediction interface
- User-friendly frontend
- Medical data input form
- Risk prediction output
- Interactive UI

---

## 🛠️ Technologies Used

- CSS
- JavaScript
- Machine Learning (backend integration possible)
